# WebGPU Life

A minimal WebGPU demo that renders a grid of cells onto a canvas using GPU instancing and bind groups.

## Files

- `index.html` - main demo page with WebGPU setup and rendering logic.

## Usage

1. Open `index.html` in a browser with WebGPU support.
2. The canvas will render the cell grid and alternate between two stored cell states.

## Requirements

- A browser with WebGPU enabled (such as Chrome Canary or Edge Canary).
- A compatible GPU and operating system.

## Notes

This project is intended as a simple WebGPU starting point for experimenting with shaders, buffer resources, and instance rendering.
